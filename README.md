# WebdriverIO test

Simple test script to run a test script against Chrome using [WebdriverIO](http://webdriver.io/guide.html).

To run:

```
npm test
```

## Requires

Tested on Mac

- JRE (for running Selenium)
- Node and npm
- Chrome
